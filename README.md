# Ticketing App

A simple Ticketing UI built using ReactJS.

Live Link: https://ticketing-app-seven-rose.vercel.app/

---

## Tech Stack

- ReactJS
- JavaScript
- Tailwind CSS
- Vercel (Deployment)

---

## Features

- Ticket list UI
- Ticket status display
- Clean card-based layout
- Responsive design
- Reusable components

---

## Installation

Clone the repository:

git clone https://github.com/ajitdevv/Ticketing-app
Install dependencies:

npm install

Run the project:

npm run dev

---

## Deployment

The project is deployed on Vercel.
